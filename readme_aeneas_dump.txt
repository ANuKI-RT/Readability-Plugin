Die Datei "ANE-OHB-aeneas-repo-r1-r1024.dump" enthält das komplette OHB AENEAS Projekt-SVN-Repository inklusive der gesamten Historie (Revision 1 vom 15.11.2016 bis Revision 1024 vom 25.06.2020).

Der Dump wurde mit "svnrdump dump" (SVN Kommandozeilentool) erzeugt und kann mit "svnadmin load" (ebenfalls ein SVN Kommandozeilentool) in ein lokales, bzw. mit "svnrdump load" in ein entferntes, leeres (!) SVN-Repository geladen werden.

Der Code ist eine Erweiterung des AENEAS Dev Environments und muss in dieses importiert werden. Der Code steht dem DLR bereits aus dem Vorgänger-Projekt zur Verfügung.
